macos上安装mysqlclient详解

安装过mysqlclient的人都知道这个东西相当难安装成功，特别是在windows和mac上面，现在我来详细讲解一下在mac上面的安装过程。

安装

第一步: 修改mysql配置文件

第一步就玩的很大，但是别担心，按着下面说的做不会出问题： 文件地址：/usr/local/bin/mysql_config 原来为：

Create options

libs=”-L$pkglibdir” libs="$libs -l " 修改为：

1
2
3

# Create options
libs="-L$pkglibdir" 
libs="$libs -lmysqlclient -lssl -lcrypto"

修改完保存退出即可.

第二步：添加环境变量

如果下面的变量地址不一致，可以使用brew info openssl查看OpenSSL安装地址。

1
2

export LDFLAGS="-L/usr/local/opt/openssl/lib"
export CPPFLAGS="-I/usr/local/opt/openssl/include"

示例：

第三步：安装mysqlclient

如果看到下面的错误：

1
2
3

ld: library not found for -lssl
    clang: error: linker command failed with exit code 1 (use -v to see invocation)
    error: command '/usr/bin/clang' failed with exit status 1

直接使用命令安装PyMySQL, 没错，没有看错也没有写错，安装它是为了解决错误：

1

pip3 install PyMySQL

之后再安装mysqlclient：

1

pip3 install mysqlclient

即可。
