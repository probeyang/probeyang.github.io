简介

本文使用pipenv安装django。

安装

第一步，安装pipenv

pip install pipenv

第二步，安装django

注意这里如果要安装django，最好新建个文件夹，在此文件夹下面安装，django会安装在此文件夹下 pipenv install django 要確認 django 有沒有安裝好，請輸入 pipenv run python -m django –version 如果出現 2.0 以上就代表你有順利的安裝好。

安装一个django项目

pipenv run django-admin startproject 项目名称 注意，这个是项目，还有个命令是pipenv run django-admin startapp app名称。这是一个项目下面的应用。

一个项目可以包含多个app，一个app可以在多个项目中。
