简介

此工具模拟浏览器（实际上是直接调用浏览器driver）截取页面，所以存在各个浏览器不同的driver情况。

使用

首先，下载selenium：

pip install selenium

再次，写代码test_sele.py：

1

from selenium import webdriver

brower = webdriver.Firefox()
url = "http://www.baidu.com"
brower.set_window_size(1200, 900)
brower.get(url)

brower.save_screenshot('baidu.png')
brower.close()

然后执行：python3 test_sele.py

这个错误是因为没有geckodriver程序，需要下载并解压后放到/usr/bin或者/usr/local/bin目录下面。下载地址：https://github.com/SeleniumHQ/selenium/blob/master/py/docs/source/index.rst 或者： https://github.com/mozilla/geckodriver/releases 我在mac上安装好后还是会报错，应该是我没有安装Firefox的原因。所以还需要安装Firefox到电脑上。

from selenium import webdriver

# brower = webdriver.Firefox()
brower = webdriver.Chrome('/Users/yang/study/python/test/driver/chromedriver')
url = "http://www.baidu.com"
brower.set_window_size(1200, 900)
brower.get(url)

brower.save_screenshot('baidu.png')
brower.close()

注意，这里Chrome()里面一定要放能使用的chromedriver，这个不是chrome程序，而是需要单独下载的driver程序。 下载地址在：https://sites.google.com/a/chromium.org/chromedriver/home或者直接在： https://sites.google.com/a/chromium.org/chromedriver/downloads下载然后放到你自己能找到的地方，并在代码中指明，我就下载在/Users/yang/study/python/test/driver/chromedriver。下载后需要解压，解压出来才是此文件。 提醒一下，使用Chrome方法必须指定chromedirver的地址。
