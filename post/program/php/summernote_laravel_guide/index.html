简介

富文本框扩展，国外比较专业的叫法是WYSIWYG editor。也就是所见即所得编辑器。现在比较不错的是summernote，而且它跟laravel兼容效果也好。其他老牌的CKeditor，Ueditor等就不介绍了。如果要对中文支持好一些的话，可以选择百度的UEditor。我觉得summernote够用，且没有太要求中文的东西，所以选择它进行开发。github地址：https://github.com/summernote/summernote

安装

你完全可以按照github中的说明文档进行安装配置使用。这里我们还是稍微赘述一下。

  1. 新建一个html文件，然后将下面的内容放到head标签中：
1
2
3
4
5

<script src="https://cdnjs.cloudflare.com/ajax/libs/blueimp-file-upload/9.22.0/js/vendor/jquery.ui.widget.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/blueimp-file-upload/9.22.0/js/jquery.iframe-transport.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/blueimp-file-upload/9.22.0/js/jquery.fileupload.min.js"></script>
<link href="https://cdnjs.cloudflare.com/ajax/libs/summernote/0.8.12/summernote-lite.css" rel="stylesheet">
<script src="https://cdnjs.cloudflare.com/ajax/libs/summernote/0.8.12/summernote-lite.js"></script>

注意：这里GitHub中的说明文档有问题，它提供的include方式有个前提：必须自己先将它的文件下载下来，而且font等文件都是按照下载下来的文件夹结构放置的，否则会有导航栏图标不显示的问题以及不显示summernote的问题。

  1. html中声明div标签 在html文件的body中添加下面的代码：
1

<div id="summernote"></div>
  1. html中的script中实例化summernote 接下来就是实例化summernote了，这里需要在html中添加script脚本，内容为： $(document).ready(function() { $(’#summernote’).summernote(); });

是不是很简单，这样刷新页面即可看到summernote编辑器了。而且图片等等功能都齐全。

配置

它基本上没有什么配置。注意两点即可： 一， 按照上面说的将js，css文件引入，如果想将js，css下载到本地进行本地引入，那么注意这里必须将font文件夹也下载下来，这里所有资料都没有讲，但这是必须的。 二， html中添加一个summernote的定位div。这样summernote才能实例化后找到位置显示。 如果想更多配置，可以参考它的官网等资料： https://summernote.org/plugins

使用

使用其实很简单，就是一句js：

1

$('#summernote').summernote();

如果想更复杂的配置和设置，可以搜索网上资料，我在以下的使用示例中，会有比较多的配置参数，可以参考。 另外中文里面有个比较不错的文档也可以参考： https://laravelacademy.org/post/9476.html

@extends('layouts.richeditor')

@section('content')
    <div class="">
        <form method="POST" action="{{ route('tutorial.save') }}">
            {{ csrf_field() }}
            <div class="col-xs-12 col-sm-12 col-md-12">
                <div class="col-xs-12 col-sm-12 col-md-12">
                    <div class="form-group">
                        <span class="required">*</span>
                        <label for="usr">教程标题:</label>
                        <input type="text" class="form-control" id="title" name="title" placeholder="请输入标题">
                    </div>
                    <div class="form-group">
                        <span class="required">*</span>
                        <label for="summernote">教程内容：</label>
                        <div id="summernote"></div>
                    </div>
                </div>
                <div class="col-xs-12 col-sm-12 col-md-12 text-center">
                    <button type="button" id="save_tutorial" class="btn btn-primary">保存</button>
                </div>
            </div>
        </form>
    </div>
    <script>
        $('#summernote').summernote({
            placeholder: '请输入教程内容',
            tabsize: 2,
            height: 500
        });

        $("#save_tutorial").click(function () {
            var title = $("#title").val();
            var html = $('#summernote').summernote('code');
            if (title == '') {
                alert("请输入教程标题！");
                return false;
            }
            if (html == '') {
                alert("请输入教程内容！");
                return false;
            }
            $.ajax({
                headers: {
                    'X-CSRF-TOKEN': $('meta[name="csrf-token"]').attr('content')
                },
                method: "POST",
                url: "{{ route('tutorial.add') }}",
                dataType: "json",
                data: {title: title, html: html},
                success: function (data) {
                    if (data.code == 200) {
                        location.reload();
                    } else {
                        alert(data.msg);
                    }
                }
            });
        });
    </script>
@endsection

@section('content')
    <div class="">
        <form method="POST" action="{{ route('tutorial.save') }}">
            {{ csrf_field() }}
            <div class="col-xs-12 col-sm-12 col-md-12">

                    </div>
                    <div class="form-group">
                        <span class="required">*</span>
                        <label for="summernote">教程内容：</label>
                        <div id="summernote_hide" style="display:none;">{{ $tutorial->content }}</div>
                        <div id="summernote"></div>
                    </div>
                </div>
                <div class="col-xs-12 col-sm-12 col-md-12 text-center">

        $('#summernote').summernote({
            focus: true,
            toolbar: [
                // ['style', ['bold', 'italic', 'underline', 'clear']],

                ['table', ['table']],
                ['misc', ['fullscreen', 'undo', 'redo']],

                    ['remove', ['removeMedia']]
                ],

                ],

                    ['add', ['addRowDown', 'addRowUp', 'addColLeft', 'addColRight']],
                    ['delete', ['deleteRow', 'deleteCol', 'deleteTable']],
                ],
                air: [
                    ['color', ['color']],
                    ['font', ['bold', 'underline', 'clear']],
                    ['para', ['ul', 'paragraph']],
                    ['table', ['table']],
                    ['insert', ['link', 'picture']]
                ]
            },
            placeholder: '请输入教程内容',
            tabsize: 2,
            height: 500
        });
        $("#summernote").summernote("code", $("#summernote_hide").html());

        $("#save_tutorial").click(function () {
            var id = $("#id").val();
