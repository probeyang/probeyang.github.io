PHPStorm配置创建文件时自动添加文件头注释方法

2019-08-09
编程语言
约 451 字 预计阅读 1 分钟 次阅读

配置入口

MAC版

PhpStorm -> Preferences… -> Editor -> File and Code Templates -> includes -> PHP File Header

打开PhpStorm->首选项->编辑器->文件和代码模板->包括->PHP文件头

windows版

File -> settings -> Editor -> File and Code Templates -> includes -> PHP File Header

文件->设置->编辑器->文件和代码模板->包括->PHP文件头

/**

我的配置内容如上，可以随意修改，具体参数可以参考下面：

${PROJECT_NAME} - 当前项目的名称

${FILE_NAME} - 将被创建的PHP文件名称

${NAME} - 在创建文件期间在新建文件对话框中指定的文件名称

${USER} - 系统的当前用户登录名称

${DATE} - 当前系统日期

${TIME} - 当前系统时间

${YEAR} - 当前年份

${MONTH} - 当前月份

${DAY} - 当前月的日期

${HOUR} - 当前时刻

${MINUTE} - 当前分钟

${PRODUCT_NAME} - 将被创建文件所在的IDE名称

${MONTH_NAME_SHORT} - 月份名称的前3个字母，例如: Jan, Feb, etc.

${MONTH_NAME_FULL} - 月份全称，例如: January, February, etc.

参考：https://www.kancloud.cn/ervinhua/phpstorm/441766

配置完成后保存应用即可在新文件创建后自动添加文件的头注释。
