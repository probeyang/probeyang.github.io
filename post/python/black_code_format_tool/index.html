注意：black对python3支持的效果不好，现在必须在python3.6以上版本才可以使用black。也正是因为支持效果不好，才有此说明文档。

安装步骤

如果已经有python3.6以后的版本则可以跳过前两步。

  1. 安装pyenv

因为black需要python3.6以后的版本，而有可能我们用的python是3.6以前的，我现在电脑上就是python2.7，python3.5两个版本的。再装一个感觉会越来越乱，所以就需要一个管理python版本的工具，pyenv就是这样的一个相当不错的工具。 git地址:

(pyenv)[https://github.com/pyenv/pyenv]

我的是macos，安装的命令是：

$ brew update
$ brew install pyenv

安装完成可以使用：

pyenv -v

检查是否安装成功。常用的命令有：cd $(pyenv root) 可以找到pyenv的安装路径；rm -rf $(pyenv root) 删除pyenv安装内容。 也可以使用 brew uninstall pyenv 卸载pyenv。

  1. 安装python3.6
pyenv install 3.6.6

mac下面安装的默认路径在：/Users/probeyang/.pyenv/versions/3.6.6/bin/python 可以使用命令：echo $(pyenv root)查看。
