理解函数带括号和不带括号分别代表什么意思:

如果只写一个函数名称myfunc而没有带括号，那么此时的myfunc仅代表一个函数对象；而当你写的是myfunc(a, b, c)时，就是告诉编译器“请执行myfunc这个函数”，意思就是说执行后的结果给变量名， 不执行的时候不会将函数结果给变量名。

示例代码

def myfunc(a, b):
    return a + b

mytest = myfunc
print(mytest)
mytest2 = myfunc(10,20)
print(mytest2)

总结

在python中，函数名加()，表示返回一个函数的执行结果，不加括号表示对函数的调用；加括号是运行函数，不加是函数引用。
