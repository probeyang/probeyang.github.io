PlantUml 是什么

PlantUml是一个支持快速绘制的开源项目.它定义了一套完整的语言用于实现UML关系图的描述，并使用强大的graphviz(https://github.com/plantuml/plantuml)图形渲染库进行UML图的生成，绘制的UML图可以直接导出为图片。

安装步骤

需要三个前提条件 1. Java环境 2. graphviz 3. VS Code中装扩展PlantUML

安装java

如果没有Java环境是不行的，使用命令查看环境中是否已经安装好Java：

java -version

如果没装好则安装一下，这里不再赘述，网上很多资料。

安装graphviz

Graphviz 是一个开源的额图片渲染库，可以将PlantUML脚本转换成图片。

两种安装方式：

  1. 终端执行
brew install graphviz
  1. 官网下载graphviz

VS Code中装扩展PlantUML

点击VS Code的左侧扩展图标，然后在搜索框输入PlantUML:

安装完成测试一下，随便找个uml文件测试：

@startuml
(*) --> "开始"
--> "导入报告"
if "获取报告xml文件" then
    ->[失败] "抛出‘报告文件不存在’错误并退出"
    ->[结束处理] (*)
else
-->[成功] "开启事务" 
    if "遍历xml文件，插入表数据，如果其中某一步失败" then
        -->[成功] "提交事务"
        --> "结束导入"
        -->[结束处理] (*)
    else
        -->[失败] "回滚事务"
        -->"抛出‘数据库入库失败’错误并退出"
        -->[结束处理] (*)
    endif
endif
@enduml

然后在VSCode界面中按command-shift-p（这是mac下面的按键，有的扩展是option+D），然后在输入框中输入：

> PlantUML

在输入的时候它会实时查询出安装的扩展程序，选择“PlantUML预览”即可看到预览的图效果：
