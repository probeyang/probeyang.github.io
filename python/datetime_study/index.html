    日期时间类操作详解

    python的日期时间操作相当混杂，一会time库，一会datetime库，网上资料又是杂乱到极点，糟糕的甚，于是我这里整理了一些常用的时间、日期转换操作等方法，供自己也供大家以后少走点弯路。

    日期转时间戳

    方法一：使用time、datetime混合

    import time
from datetime import datetime

time_now = '2018-09-07 21:10:08'

t = datetime.strptime(time_now, '%Y-%m-%d %H:%M:%S')
timestamp = int(time.mktime(t.timetuple()))
# timestamp = int(time.mktime(datetime.strptime(time_now, '%Y-%m-%d %H:%M:%S').timetuple()))
print(timestamp)

    方法二：只使用time实现（推荐使用此方式）

    import time

time_now = '2018-09-07 21:10:08'

timestamp = int(time.mktime(time.strptime(time_now, '%Y-%m-%d %H:%M:%S')))

print(timestamp)

    时间戳转日期

    当前时间转日期

    from datetime import datetime

print(datetime.now().strftime('%Y-%m-%d %H:%M:%S'))

    任意时间戳转日期

    import time

timestamp = 1536325808

st = time.localtime(timestamp)

date_time = time.strftime('%Y-%m-%d %H:%M:%S', st)
print(date_time)

    获取当前时间戳

    当前时间戳

    import time

d = time.time()

print(d)

    1536330821.832002

    当前时间戳10位

    import time

d = int(time.time())

print(d)

    1536330821

    当前时间戳13位

    import time

d = int(round(time. time() * 1000))

print(d)

    1536331274726
