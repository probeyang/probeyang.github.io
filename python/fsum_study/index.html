    使用math中的fsum进行数据求和汇总

    fsum这个方法可以用于进行可迭代数据的求和或者汇总。避免了还需要些循环语句。但通常情况下在不知道可迭代数据中的每个元素都是int或者float类型的时候，我们还是需要使用循环对其每个元素进行遍历检查，防止fsum函数报错。

    使用方法

    # 方法一 使用math的fsum
from math import fsum

a = ['11', '33', 33, 44, 55, 66]
b = fsum([i if isinstance(i, float) else float(i) for i in a[0:4]])
print(b)

    输出为： 

    /Library/Frameworks/Python.framework/Versions/3.5/bin/python3.5 /Users/probeyang/study/python/test/test8.py
121.0


    # 方法二 直接循环操作
sum_data = 0
for i in a[0:4]:
    sum_data += i if isinstance(i, float) else float(i)

print(sum_data)

    输出为： 

    /Library/Frameworks/Python.framework/Versions/3.5/bin/python3.5 /Users/probeyang/study/python/test/test8.py
121.0

    综上可见，方法一代码更pythonic，但是性能上比方法二差，方法二更直接，没有方法一优雅。然而一般情况下我们应该都选方法二处理。只有在知道每个元素都是int或者float的时候fsum才能发挥其威力。
